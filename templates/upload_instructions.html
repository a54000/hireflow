Bulk Upload Instructions

Review this checklist before opening the upload screen. It helps prevent blank-template uploads, wrong requirement mapping, and missing CV matches.

Download Spreadsheet Template Open Bulk Upload
Upload the completed candidate spreadsheet, not the blank template.

1Prepare Requirements

  1. Create the Requirement in ATS before bulk upload.
  2. Use the same Requirement title in the spreadsheet.
  3. For multiple roles, keep each candidate row mapped to the correct Requirement.

2Prepare Spreadsheet

  1. Download the template and fill candidate details below the hint row.
  2. Do not rename or remove the required columns.
  3. Save the completed file with a clear name, for example L&D.xlsx.

3Attach JD Files

  1. Attach one JD file for each Requirement included in the spreadsheet.
  2. Name the JD file so it includes the Requirement title or a clear abbreviation.
  3. Example: Learning Development JD.pdf or JD_LD_Taggd.pdf.

4Attach CV Files

  1. Attach CVs for all candidates in the spreadsheet.
  2. CVs are matched by email, phone, or candidate name.
  3. Use candidate names in CV filenames when possible.

Before Clicking Upload Excel

CheckWhat to confirm
JD filesThe selected JD files are for the same Requirements used in the spreadsheet.
